# Fintech API Automation Framework

## Project Overview
This project demonstrates API automation using Rest Assured, TestNG, and Maven.  
It simulates real-world booking workflows similar to fintech transaction flows.

## Tech Stack
- Java
- Rest Assured
- TestNG
- Maven
- JSON
- GitHub

## Test Coverage

✔ Create Booking (POST)  
✔ Get Booking (GET)  
✔ Generate Auth Token  
✔ Update Booking (PUT)

## Features

- Dynamic bookingId handling
- Token-based authentication
- Request & response logging
- Modular payload design
- Config-based base URL
